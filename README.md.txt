# Hotel Revenue Performance Analysis (YoY)

## Business Problem
Hotel management needs to monitor occupancy, pricing (ADR), and revenue performance over time to identify trends, compare year-over-year performance, and support pricing and capacity decisions.

## Dataset / Data Source
Transformed hotel operational and revenue data prepared in Power BI using standard hospitality KPIs (Occupancy %, ADR, RevPAR, Total Revenue) for analytical and dashboarding purposes.

## Key Metrics Explained
- **Occupancy %** = Rooms Occupied / Total Rooms  
- **ADR (Average Daily Rate)** = Room Revenue / Rooms Sold  
- **RevPAR** = ADR Ã— Occupancy %  
- **Total Revenue** = Room Revenue across the period  

## Summary of Analysis
- Presented an executive summary of 2025 hotel revenue and occupancy performance.
- Compared key KPIs and revenue trends against 2024.
- Provided actionable insights on seasonal revenue patterns and hotel-type performance.

## Tools Used
- Power BI (Data modeling, DAX calculations, Visualizations)
